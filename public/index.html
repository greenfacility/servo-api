Welcome to the Greenfacilites API Documentation

Users Api

These are the APIs for the users

GET REQUEST: /users/api
x-auth-token: header with token from login to get user

GET REQUEST: /users/api/all
To get all the users from the server

PUT REQUEST: /users/api
To Register, Need JSON Like sample below 

                
        {
            "firstname": "Test",
            "lastname": "LastName",
            "password": "password",
            "email": "email@domain.com",
            "phonenumber": 0123456789,
            "address": "Sample Avenue, City, State",
            "picture": ""
        }

POST REQUEST: /users/api
To login, Need JSON Like sample below 

                
        {
            "email": "email@domain.com"
            "password": "password",
        }
